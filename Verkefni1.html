VEFSÍÐUGERÐ

Í þessu fyrsta verkefni munum við búa til vefsíðu með því að nota HTML-sniðmat. Í verkefninu mun ég útskýra ferlið skref fyrir skref við gerð vefsíðunnar og gera grein fyrir því sem ég lærði á leiðinni.

UNDIRBÚNINGUR

Ég byrjaði með því að horfa á kennslumyndband fyrir fyrsta verkefnið, sem hjálpaði mér að átta mig á verkferlinu og komast fljótt af stað. Að því loknu skoðaði ég sniðmát á HTML5 UP og valdi Helios sem grunn að vefsíðunni. Til að vinna með kóðann notaði ég Brackets vegna rauntímaforskoðunar (live preview), sem gerði mér kleift að sjá breytingar strax og fínstilla útlit og virkni eftir þörfum. Að auki setti ég upp Git Bash til að halda utan um verkefnið og vinna með Git.

UPPSETNING Á GITHUB

Ég bjó til nýtt repository á GitHubinu mínu sem heitir VinnubokAaron. Mjög auðvelt er að búa til aðganginn sjálfann en þegar það er búið þarf að búa til vefslóð fyrir vefsíðuna sjálfa.


Ég bjó til nýtt repository á GitHub og tengdi síðan verkefnamöppuna á tölvunni við repository-ið með Git, Opnaði Git Bash í verkefnamöppunni og Svo keyri ég þessa skipun til að geta fikrað mig áfram github megin 



echo "#VinnubokAaron" >> README.md
git init 
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/aaronasug/VinnubokAaron.git
git push -u origin main

Þegar verkefnamappan var komin inn á GitHub var hægt að virkja GitHub Pages og fá þar með vefslóð að síðunni, þannig að vefsíðan varð aðgengileg á netinu í gegnum link sem GitHub útbýr.
